# 🏥 Hospital Management System

A Hospital Management System developed using **Python** to manage patient records, doctor details, appointments, and hospital operations efficiently.

---

## 📌 Project Overview

This project is a console-based Hospital Management System designed to automate and simplify hospital administrative tasks. It helps manage patient data, doctor information, appointments, and billing in an organized manner.

---

## 🚀 Features

- 👨‍⚕️ Add / View / Update / Delete Patient Records  
- 🩺 Manage Doctor Information  
- 📅 Appointment Scheduling  
- 💊 Treatment & Billing Management  
- 📂 Data Storage and Retrieval  
- 🖥 User-Friendly Console Interface
